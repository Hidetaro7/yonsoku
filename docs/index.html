ジャケットを猫で埋め尽くす

みなさんこんにちは、Twitterのちくわ（@chikuwa_cats）です。

ワタクシ、ちくわは音楽ユニット、ニューロナのリーダーをやっています。
飼い主がちくわのために曲を書いてくれました、その名も「四足のヒーロー」です。
社会や対人関係に苦しむニンゲンという生き物を見て、もっと猫のように自由に、そして依存せずに時にはツンと突き放すことこそ愛だと語るそんなロックチューンの元気な曲です。

今回「四足のヒーロー」をシングル・カットして（AppleやGoogleやAmazonなどの）各種音楽ストアでリリースしたいと思います。そうなると音源のジャケット画像が必要です。
いっしょにジャケットを埋め尽くしませんか？

募集、あなたの愛猫の写真

こんなジャケットをイメージしています。（100匹の場合、あくまで予定です）

本当に100匹参加したら、小さすぎてよくわからないかもしれません。 でも、応募数が少なすぎた場合大きく見えるかもしれません、それは正直やってみないとわかりません。
